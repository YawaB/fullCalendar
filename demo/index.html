EasyCal

A lightweight, zero-dependency JavaScript calendar library. Drop it in, style it your way.

📦 ~24 KB 🚫 Zero dependencies 🗓 4 views ✨ Vanilla JS 📋 Event API
Live Demo
Click an event or date to see callback output here.
Usage 
// 1. Include the CSS and JS
<link rel="stylesheet" href="easycal.css">
<script src="easycal.js"></script>

// 2. Add a container element
<div id="calendar"></div>

// 3. Initialize
const cal = new EasyCal('#calendar', {
  view: 'month',
  events: [
    {
      title: 'Team Standup',
      start: '2025-01-15T09:00:00',
      end:   '2025-01-15T09:30:00',
      color: '#3b82f6'
    }
  ]
});
