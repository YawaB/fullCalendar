EasyCal

A zero-dependency JavaScript calendar library. Four views, fully themeable, 24 KB flat. Drop it in and own it.

Quick Start

// 1. Include CSS + JS
<link rel="stylesheet" href="easycal.css">
<script src="easycal.js"></script>

// 2. Add a container
<div id="calendar"></div>

// 3. Initialise
const cal = new EasyCal('#calendar', {
  view: 'month',
  events: [
    {
      title: 'Team Standup',
      start: '2025-01-15T09:00',
      end:   '2025-01-15T09:30',
      color: '#3b82f6',
    }
  ],
});

Features

🗓

4 Views

Timeline month, week, day, and list/agenda — all built in.

⚡

Zero Dependencies

No jQuery, no Moment.js, no lodash. Pure vanilla JS.

🎨

CSS Variables

Fully themeable via CSS custom properties. No Sass required.

📡

Event API

Add, remove, and update events dynamically at runtime.

🌍

i18n Ready

Locale-aware formatting via the native Intl API.

📦

npm Ready

Ships UMD + ESM. Works with any modern bundler.
